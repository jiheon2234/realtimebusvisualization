# real_time_bus.ipynb

## 내가 자주타는 버스위치 실시간 시각화해보기

> 파이썬과 대전버스 API를 활용하여 내가 자주 타는 3종류 버스 위치를  
> 실시간으로 시각화하기
>
> > 1초마다 모든 버스의 위치를 응답받고, 이를 각각 객체로 만들어  
> > 지도에 업데이트

## 개발환경

| 환경 | 이름                                                                          |
| ---- | ----------------------------------------------------------------------------- |
| 작성 | Colab                                                                         |
| API  | [대전버스](https://www.data.go.kr/data/15058247/openapi.do?recommendDataYn=Y) |
| 언어 | python                                                                        |
